🖥️ Sistema Web – Projeto de TCC

Este repositório contém o código desenvolvido para o meu Trabalho de Conclusão de Curso (TCC) do ensino médio. O projeto consiste em uma aplicação web construída utilizando Java, executada em Apache Tomcat e integrada a um banco de dados MySQL.

Embora tenha sido desenvolvido em uma fase inicial da minha jornada na programação, mantenho este projeto como registro da minha evolução como desenvolvedor e como exemplo de aprendizado prático no desenvolvimento de aplicações web.

🚀 Funcionalidades

🌐 Aplicação Web: Sistema executado em servidor Apache Tomcat.

🗄️ Integração com Banco de Dados: Armazenamento e manipulação de dados utilizando MySQL.

📊 Gerenciamento de Dados: Operações básicas de manipulação e consulta de informações.

🔐 Estrutura de Backend: Implementação da lógica da aplicação utilizando Java.

🛠️ Tecnologias

Java

Apache Tomcat

MySQL

MySQL Connector/J

HTML / CSS
